Reports with Watermark

Video Tutorial

Aktiv Software has prepared the "Reports with Watermark" module with the aim of adding Watermark to all printed reports of Odoo with all four template layout. Moreover, this module comes with various customizations when user enables the option of adding watermark in Odoo.

Below mentioned are the features of this module:

  • You can use all watermark features with odoo's basic four layout.
  • You can add Company's logo as Watermark.
  • You can also add a background image as Watermark.
  • You can input text as watermark and customize its color, size and angle.

Prerequisites before Installation

No You just need to configure document layout from General Settings because this module works for Standard Layout,Boxed Layout, Background Layout and Clean Layout.

This module is working with wkhtmltopdf version 0.12.2

Configurations

  • Go to Settings
  • Select User & Companies
  • Configure Watermark settings

Reports with Watermark Menu

Once you have configured the settings of watermark you need to click on the checkbox of Watermark provided below Company Tagline. As soon as you click on the watermark option other customization options will be activated as shown in the below image.

Select Company Logo

Click on the Watermark Option dropdown and select "Company Logo". Selecting this attribute will allow you to add your company's logo as Watermark in report. Let us consider adding Aktiv's logo to the Company logo and try to generate report with watermark.

Now, let us consider adding text to the report as Watermark by selecting "Company's name" attribute. You can set appropriate rotation angle, font size and color as per your requirements. Here, we set Rotate as true and angle or roatation is 45 and size of font is 45 with Red color".
